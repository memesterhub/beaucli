# Welcome to beaucli documenatation

## Installation

by pip
```sh
pip install beaucli
```
from source code
```sh
git clone https://github/com/memesterhub/beaucli
python setup.py install
```

## Methods

* `beaucli.fore` - Foreground shell colors/styles.
* `beaucli.back` - Background shell colors/styles.
